# Laundry Mart Website

This project is a simple website created using HTML. It represents a laundry service called **Laundry Mart**.
The website includes an about section, list of services, a price table, and a booking form where users can enter their details.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Find the **index.html** file.
4. Double click the file or open it with any web browser.
5. The Laundry Mart website will open in your browser.
