# CSS Styling Practice Task

This project is a simple HTML and CSS practice task.
The goal of this task is to understand how CSS works with HTML using **classes, ids, and element selectors**.

In this task, different heading tags and paragraphs are styled using CSS. Background colors, text colors, and font sizes are applied based on the instructions provided.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Make sure both **index.html** and **style.css** files are in the same folder.
4. Open the **index.html** file in any web browser.
5. The styled webpage will appear according to the CSS rules.
